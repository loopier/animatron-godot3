# Keyboard Shortcuts ## Editor Keys | Description --------|------ `CTRL + H` | show this help and OSC commands. `CTRL + ENTER` | evaluate current block. `SHIFT + ENTER` | evaluate current line. `CTRL + E` | toggle editor and post window. `CTRL + SHIFT + E` | toggle editor. `CTRL + P` | toggle post window. `CTRL + SHIFT + P` | clear post window. `CTRL + O` | open a file in the text editor. `CTRL + S` | save the contents of the text editor to a file. `CTRL + D` | duplicate line. `CTRL + +` | increase font `CTRL + -` | increase font
